# Portfólio Pessoal - Luiz Felipe

Este é o meu portfólio pessoal e profissional, desenvolvido como parte dos critérios de avaliação da disciplina de **Programação para a Internet I**, ministrada pelo Professor Ely no IFPI. 

O design do projeto foi construído com uma abordagem moderna, limpa e totalmente responsiva, simulando a página profissional de um Analista de Sistemas / Estudante de Tecnologia, utilizando uma paleta em tons de azul e foco na usabilidade em dispositivos móveis.

## 🎯 Tema Escolhido
Estudante de Tecnologia / Analista de Sistemas

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo e acessibilidade básica.
* **CSS3:** Estilização avançada pura, layouts responsivos (Flexbox e CSS Grid), variáveis nativas (`:root`) e interações dinâmicas (efeitos Hover).

## 🛠️ Funcionalidades e Critérios Atendidos
* **Semântica HTML:** Uso correto de tags como `header`, `nav`, `main`, `section`, `article` e `footer`.
* **Responsividade (Mobile-First):** Layout adaptável para celulares através de *Media Queries*, com cards empilhados e imagens responsivas.
* **Restrições Respeitadas:** Desenvolvimento 100% puro, sem a utilização de frameworks como Bootstrap ou Tailwind.
* **Componentes Obrigatórios:** Inclusão de um formulário de contato estruturado, listas de habilidades e uma tabela de trajetória acadêmica.
* **Acessibilidade e SEO:** Organização hierárquica de títulos (h1, h2, h3) e links internos funcionais para navegação suave.

## 🚧 Principais Desafios Encontrados
* Estruturar a responsividade da grade de projetos para se transformar de um grid de três colunas em um fluxo vertical suave em telas menores, sem quebrar os componentes.
* Criar um visual moderno (com sombras flutuantes, bordas arredondadas e pílulas de habilidades) mantendo a restrição de usar apenas CSS puro.
